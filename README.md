# ☕ Web Services com Spring Boot e JPA

> 💻 API REST desenvolvida com Java, Spring Boot e JPA / Hibernate

---

## 🏢 Sobre o projeto

Projeto backend desenvolvido durante um Workshop ministrado pelo instrutor Nelio Alves.

A aplicação consiste na criação de uma API REST utilizando Spring Boot, com persistência de dados através do JPA / Hibernate, seguindo boas práticas de desenvolvimento em camadas.

O sistema permite operações de CRUD (Create, Retrieve, Update, Delete) e simula um ambiente real de aplicação backend. :contentReference[oaicite:0]{index=0}

---

## ⚙️ Funcionalidades

- Criação de API REST com Spring Boot  
- Operações CRUD (Create, Read, Update, Delete)  
- Persistência de dados com JPA / Hibernate  
- Estrutura em camadas (resource, service, repository)  
- Integração com banco de dados (H2 e/ou PostgreSQL)  
- Tratamento de exceções  

---

## 🛠️ Tecnologias utilizadas

- Java  
- Spring Boot  
- JPA / Hibernate  
- Maven  
- Banco de dados H2 (teste)  
- PostgreSQL (produção)
